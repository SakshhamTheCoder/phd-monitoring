## Getting Started

1. Install dependencies:
    ```sh
    npm install
    ```
2. Start the server:
    ```sh
    npm run dev
    ```

## Structure

-   `app/` - Controllers, middleware, services, etc.
-   `routes/` - Route definitions (API, web, etc.)
-   `config/` - Configuration files
-   `database/` - Database models, migrations, seeders
-   `public/` - Static files
-   `resources/` - Views, assets
-   `storage/` - File storage
